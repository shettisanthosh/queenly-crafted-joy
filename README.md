# 👑 Queens Puzzle Game

## About the Project

This project is my implementation of the **Queens Puzzle Game**, inspired by the popular logic game available on platforms like LinkedIn. The goal wasn't just to recreate the game but to build a polished, interactive experience that feels smooth, intuitive, and enjoyable to play.

While developing this project, I focused on writing clean, modular code and designing the application in a way that makes it easy to maintain and extend with new features in the future.

---

## How to Play

The objective is simple, but solving each puzzle requires careful logical thinking.

You need to place queens on the board while following four rules:

- Only **one queen** can be placed in each **row**.
- Only **one queen** can be placed in each **column**.
- Only **one queen** can be placed inside each **colored region**.
- Queens cannot touch each other, even diagonally.

Every puzzle has a valid solution that can be reached using logic alone.

## Features

The application includes everything needed for a comfortable gameplay experience:

- Multiple board sizes
- Interactive puzzle board
- Color-coded regions
- Queen and X marker placement
- Real-time rule validation
- Conflict highlighting
- Automatic win detection
- Timer and move counter
- Undo and redo functionality
- Restart puzzle
- Responsive design for desktop and mobile
- Smooth animations
- Dark and Light mode
- Automatic progress saving

## Running the Project

Clone the repository:

```bash
git clone https://github.com/your-username/queens-game.git
```

Move into the project directory:

```bash
cd queens-game
```

Install the required dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
